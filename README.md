# SKY职忆

**SKY职忆** 是一个面向大学生求职场景的 AI 求职辅助平台。

它的核心目标很简单：

> 把学生零散的简历、项目、实习、竞赛和面试反馈，整理成可以长期复用的“求职记忆库”，再用于岗位匹配和面试训练。

---

## 这个项目是做什么的？

很多大学生不是没有经历，而是：

- 不知道怎么把经历写进简历；
- 投不同岗位时要反复改同一段经历；
- 不知道哪些岗位更适合自己；
- 面试前不知道会被追问什么；
- 面试后的反馈没有沉淀，下次还会重复踩坑。

**SKY职忆** 想解决的就是这些问题。

用户只需要导入简历或粘贴经历，系统就可以完成：

```text
材料导入 → 经历解析 → 记忆精炼 → 岗位匹配 → 面试训练 → 反馈回写
```

---

## 主要功能

### 1. 简历 / 经历导入

支持用户上传或粘贴自己的求职材料，包括：

- 简历
- 项目经历
- 实习经历
- 竞赛经历
- 技能描述
- 面试反馈

支持文件格式：

```text
txt / md / pdf / docx
```

---

### 2. AI 经历解析

系统会从用户材料中自动提取：

- 教育背景
- 目标岗位
- 技能关键词
- 项目经历
- 实习经历
- 竞赛经历
- 需要补充的信息

页面会用标签和卡片展示解析结果，不会直接输出一大段代码。

---

### 3. 求职记忆库

系统会把用户经历整理成结构化记忆，例如：

- 做过什么项目；
- 会哪些技能；
- 适合什么岗位；
- 哪些经历适合写进简历；
- 面试中有哪些反馈。

这些内容可以长期保存，并在后续岗位匹配和面试训练中重复使用。

---

### 4. 岗位匹配

系统内置本地校招岗位库，可以根据：

- 岗位关键词
- 城市偏好
- 用户经历标签

推荐匹配度较高的岗位。

岗位以卡片形式展示，包括：

- 公司名称
- 企业类型
- 匹配度
- 岗位信息
- 工作地点
- 投递链接

---

### 5. 面试训练

系统可以根据用户的求职记忆和目标岗位生成：

- 1 分钟自我介绍
- 常见面试问题
- 岗位相关问题
- 项目追问题

用户可以输入自己的回答，系统会给出改进建议，并可以把回答保存到记忆库中。

## 如何运行

### 1. 配置 DeepSeek API Key

打开：

```text
backend/.env
```

填写你的 DeepSeek API Key：

```text
DEEPSEEK_API_KEY=你的 API Key
DEEPSEEK_MODEL=deepseek-chat
DEEPSEEK_BASE_URL=https://api.deepseek.com/chat/completions
```

---

### 2. 安装依赖

进入后端目录：

```bash
cd backend
pip install -r requirements.txt
```

---

### 3. 启动项目

```bash
python server.py
```

Windows 用户也可以直接双击：

```text
backend/run_backend.bat
```

---

### 4. 打开网页

浏览器访问：

```text
http://127.0.0.1:5000
```

注意：请先启动后端，再打开网页。

## 项目亮点

- 不是一次性改简历，而是建立可复用的求职记忆库；
- 同一段经历可以面向不同岗位重复使用；
- 岗位匹配和面试训练都基于用户自己的经历；
- 面试反馈可以继续写回记忆库；
- 适合大学生求职、实习投递和面试准备。
